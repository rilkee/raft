### 什么是Raft一致性算法
一致性算法用来保持集群中多个节点之前状态的一致性，相比其他一致性算法，raft更加通俗易懂。
其Paper中阐述了Raft的三个主要特点是：
1. leader election
2. log replication
3. safety
